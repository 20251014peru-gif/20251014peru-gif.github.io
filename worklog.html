파일링크 사용법
💡 최초 1회 설정 필요 — 아래 내용을 메모장에 붙여넣고 localfile.reg로 저장 후 더블클릭하세요.
Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\localfile]
"URL Protocol"=""
@="URL:Local File Protocol"

[HKEY_CLASSES_ROOT\localfile\shell\open\command]
@="cmd /c start \"\" \"%1\""
등록 후, 업무·메모·회의·전달 항목에 파일 또는 폴더 경로를 입력하면 클릭 한 번으로 열 수 있습니다.
예시: C:\서희타워\품의서\26-5-2.docx 또는 D:\업무\
진단
백업 / 복구

백업은 사진까지 포함한 전체 데이터를 JSON 파일로 저장합니다. 복구는 백업 파일을 불러와 기존 데이터에 합칩니다(같은 항목은 덮어씀).

CSV 내보내기 (엑셀)

항목별로 엑셀에서 바로 열 수 있는 CSV 파일을 내보냅니다. 사진은 제외됩니다.

AI 대화 · 분석

업무일지 데이터를 바탕으로 질문하거나 분석을 받을 수 있습니다. API 키는 이 브라우저에만 저장되며 코드/서버로 전송되지 않습니다.
